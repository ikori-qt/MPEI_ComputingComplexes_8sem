Original second assignment given:
    1. Реализовать родительский класс первичного оборудования и при необходимости его наследников
в виде конкретных видов оборудования (шина и т.д.).
    2. Реализовать класс короткого замыкания.
    3. Реализовать класс защиты.
    4. Проверить работоспособность разработанной программы нужно с помощью 10 итераций,
которые реализовать с использованием цикла while.
    5. С помощью библиотеки logging реализовать журнал событий.
    Задание включает в себя построение подстанции (для варианта 7) 220/10 кВ с 4мя линиями ВН и
2 мя линиями НН. Схема распределительного устройства представляется шиной + линией (без сложных конфигураций).
При количесвте отходящих линий больше 1 необходимо иметь секционированный выключатель.
    Модель должна включать в себя:
    - объекты первичного оборудования: линии, силовые трансформаторы, выключатели, шины
    - объекты защит: основная, резервная (на элементах 330кВ и выше предусматриваются 2 комплекта защиты)
    - объекты повреждений: трехфазное, однофазное, двухфазное и витковое КЗ
